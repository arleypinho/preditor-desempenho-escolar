Bem-Vindo ao Sistema de Predição de Alunos


Este sistema foi desenvolvido para auxiliar educadores a identificar se um aluno tem chances de ser Aprovado ou Reprovado com base em várias características e comportamentos relacionados ao desempenho acadêmico. O objetivo principal é ajudar a identificar em qual categoria um aluno provavelmente se enquadra, usando informações demográficas, atividades extracurriculares e outros fatores relacionados ao apoio acadêmico e social.
• 1: Para alunos como possíveis "Aprovado".
• 0: Para alunos como possíveis "Reprovado".

Para isso, basta preencher os campos abaixo com os dados do aluno e clicar em "Prever" para ver o "Resultado".
